# DATABASE
Academic Project at University
Project Name:
Restaurant and Bar Reservation Management System

Description:
An online system for managing reservations, orders, payments, and food delivery in restaurants and bars. It improves table organization, staff performance, and provides analytics for business management.

Technologies Used:
Microsoft SQL Server

Key Features:
Online table reservations and capacity management
Real-time orders and online payments
Loyalty system and rewards for regular customers
Management of orders, staff, and inventory
Reports and analytics for restaurant performance
Food delivery and takeaway options

What I Learned:
Designing databases and creating tables with SQL
Using DDL, DML, triggers, indexes, views, and functions in SQL
Data security and backup strategies
Integrating backend with frontend to build a fully functional system
